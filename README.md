# Projet-python-sur-les-donnees-NBA
Ce projet présente le développement d’une application web interactive en Python dédiée à l’analyse des performances NBA. Basée sur un large jeu de données, elle combine statistiques descriptives, analyses multivariées, clustering et modélisation prédictive au sein d’un dashboard Streamlit, offrant un outil d’aide à la décision clair et exploitable.
